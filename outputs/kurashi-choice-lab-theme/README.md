# Kurashi Choice Lab WordPress Theme

静的HTMLサイト「暮らしの選び方ラボ」をWordPressテーマ化したものです。

## アップロード方法

1. `kurashi-choice-lab-theme` フォルダをZIP圧縮します。
2. WordPress管理画面で `外観 > テーマ > 新規追加 > テーマのアップロード` を開きます。
3. 作成したZIPファイルを選択し、インストールします。
4. インストール後に `有効化` をクリックします。
5. `外観 > メニュー` でメインメニューとフッターメニューを作成し、表示位置に割り当てます。
6. `外観 > ウィジェット` でサイドバーやフッターウィジェットを設定します。
7. 投稿にカテゴリーとアイキャッチ画像を設定すると、記事一覧・関連記事・アーカイブに反映されます。

## SEO設定

- WordPress標準のタイトルタグに対応しています。
- 投稿・固定ページの抜粋文をmeta descriptionとして出力します。
- `_kcl_meta_description` というカスタムフィールドを設定すると、その値をmeta descriptionとして優先します。
- canonical、OGPの基本タグをテーマ側で出力します。

## 運用メモ

- トップページは `front-page.php` を使用します。
- 投稿一覧ページを別に作る場合は、WordPressの `設定 > 表示設定` で投稿ページを指定してください。
- アフィリエイトリンクは投稿本文内で通常のリンクとして管理できます。
